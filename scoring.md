# Wimbledon 2026 Prediction Scoring

**Last Updated:** 2026-07-02

## Scoring rules

### Base points — correct winner pick

| Round | Points |
|-------|--------|
| R1 | 1 |
| R2 | 2 |
| R3 | 4 |
| R4 | 8 |
| QF | 16 |
| SF | 32 |
| F | 64 |

A pick is correct if the player you predicted to win that round actually wins that match.

### Bonus points — correct score prediction

If you fill in a score prediction in predictions.md, the following bonuses apply on top of the base points:

| Accuracy | Bonus |
|----------|-------|
| Exact set score match (e.g., 6-4, 7-5, 6-3) | +5 per round |
| Correct number of sets (e.g., predicted 3 sets, match went 3 sets) | +2 per round |

### Confidence multiplier

Applies to base points only (not score bonus).

| Confidence | Multiplier |
|------------|------------|
| H (High) | x1.5 |
| M (Medium) | x1.0 |
| L (Low) | x0.5 |

Multiplier only activates on **correct** picks. Wrong picks score 0 regardless of confidence.

### Maximum possible score
